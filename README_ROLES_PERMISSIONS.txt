SxS The Unreal - Role Permissions

Rank order:
1. Leader
2. Deputy
3. Officer
4. Member

Community User:
- Not on the Guild Roster
- Can sign in and manage their own basic account
- Future: can favorite/comment when those features are enabled

Member:
- Appears on Guild Roster
- Can edit own in-game name and class
- Can create/edit own builds when community builds are enabled

Officer:
- Everything Member can do
- Can access Admin tools
- Can manage/moderate community builds
- Can moderate comments when comments are enabled

Deputy:
- Everything Officer can do
- Can manage members
- Can verify rank requests below Leader
- Can feature builds
- Can view audit log

Leader:
- Full access
- Can manage members
- Can manage ranks/permissions
- Can manage site settings
- Can manage guides when guide editing is built
- Can view audit log

Important:
Leader, Deputy, and Officer require the rank verification password during signup.
The current development password is stored in Cloudflare as RANK_VERIFICATION_PASSWORD.
Do not hardcode it in public files.
