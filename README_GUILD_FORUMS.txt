Guild Forums update

- recruitment.html is now Guild Forums.
- Starting forums included: Conqueror, Guardian, Destroyer, Dominator, The Game, and Away / Unable to Attend.
- You can create new forums from the page after entering as Officer or Admin.
- You can delete forums after entering as Admin.
- Demo Admin/Officer PIN: unrealadmin

Important:
This is a GitHub Pages friendly front-end version. It stores forums and posts in the current browser using localStorage. That means it is good for testing the layout and workflow, but it is not a real public forum database yet.

For real member logins and permissions, the next step should be Firebase Auth or Supabase Auth with member/officer/admin roles.
