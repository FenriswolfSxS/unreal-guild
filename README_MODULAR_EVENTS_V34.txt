SxS The Unreal — Modular Events V34

The Events page now uses the same structured page builder as Home and Guides.

Event Card fields:
- Event label
- Event title and description
- Start date/time
- Timezone label
- Repeat: none, daily, weekly, every two days, or monthly
- Duration in minutes
- Status
- Optional event link and link text

Use Edit: On, select a content area, and click Event Card. Event cards can be moved, duplicated, deleted, and arranged Full Width, Side by Side, or Three Across. Save Page stores the complete event board in D1.

No SQL reset is required. Keep the DB and R2_ASSETS bindings.
